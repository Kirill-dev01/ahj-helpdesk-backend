# HelpDesk (Backend)

Серверная часть для проекта управления тикетами (HelpDesk). Написана на Node.js с использованием фреймворка Koa.js. 
Сервер хранит список тикетов и обрабатывает REST API запросы в формате JSON.

## Запуск проекта локально

1. Установите зависимости:
   ```bash
   npm install.

2. Запустите сервер:
   ```bash
   node server.js.
